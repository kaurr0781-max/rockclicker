# Rock Clicker 🪨

A Cookie Clicker-inspired incremental game where you click rocks to earn points! Game state is stored server-side to prevent cheating.

## Features

- 🪨 Click rocks to earn currency
- 🏢 Purchase buildings to generate passive income
- ⬆️ Unlock upgrades to boost efficiency
- 🏆 Progress system with milestones
- 🔐 Server-side state management (no localStorage cheating!)
- 🎨 Beautiful Cookie Clicker-style UI

## Installation

```bash
npm install
```

## Running the Game

```bash
node server.js
```

Then open `http://localhost:3000` in your browser.

## Game Mechanics

- **Click**: Earn 1 rock per click
- **Buildings**: Purchase with rocks to generate passive income
- **Upgrades**: Boost your clicking power and building efficiency
- **Prestige**: (Coming soon) Reset progress for permanent bonuses
